# pygamece_practice
ts is my school coding club project

# how to use ts
1. download the `.zip` file and extract the files
2. put `pygame` directory to Desktop
3. run powershell in `pygame` directory and run game_main.py

# development environment and library
- **Language:** Python 3.14
- **Library:** [pygame-ce](https://github.com/pygame-community/pygame-ce) (LGPL-2.1)
